# Longest Common Subsequence (LCS)

A Python implementation of the Longest Common Subsequence problem with multiple algorithmic approaches.

## Project Structure

- `src/` - Source code implementations
- `data/` - Test data and inputs  
- `results_*.csv` - Performance comparison results

## Implementations

- Standard Dynamic Programming
- Optimized Dynamic Programming
- Backtracking approach
- Brute Force approach

## Usage

Run the main script:
```bash
python src/main.py
```

Run tests:
```bash
python Testing.py
```
